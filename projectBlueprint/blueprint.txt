# Institutional-Style OI Dashboard (HTML + FastAPI)

## Project Goal

Build a professional low-latency HTML dashboard for:

* ATM live monitoring
* ITM/OTM monitoring
* Multi-timeframe OI analysis
* Greeks analysis (Delta, Gamma, IV)
* Historical replay
* Live market monitoring

The project replaces Streamlit with:

* FastAPI backend
* HTML + Bootstrap frontend
* JavaScript live updates
* SQLite historical storage

---

# Why Move From Streamlit?

## Current Streamlit Issues

### Full Script Rerun

Every refresh:

* entire Python script reruns
* UI redraws completely
* more CPU usage
* slower response

### Harder UI Customization

* limited layout flexibility
* difficult realtime animations
* difficult professional trading layout

### Scaling Issues

As project grows:

* Greeks
* charts
* heatmaps
* multiple pages
* replay mode

Streamlit becomes harder to manage.

# Database Design

## Table Structure

```sql
CREATE TABLE oi_snapshots (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp TEXT,
    strike REAL,
    spot REAL,

    call_oi INTEGER,
    put_oi INTEGER,

    call_price REAL,
    put_price REAL,

    call_volume INTEGER,
    put_volume INTEGER,

    call_delta REAL,
    put_delta REAL,

    call_gamma REAL,
    put_gamma REAL,

    call_iv REAL,
    put_iv REAL
);
```

---

# IMPORTANT STORAGE RULE

## Store ONLY 5-Minute Raw Data

Correct architecture:

```text
Store:
5m snapshots only
```

Do NOT separately store:

* 15m
* 30m
* 1hr

These must be calculated dynamically.

# Gamma Interpretation Rules

| Gamma       | Meaning          |
| ----------- | ---------------- |
| < 0.004     | Slow market      |
| 0.004–0.008 | Healthy trend    |
| 0.008–0.012 | Strong expansion |
| > 0.012     | Explosive        |

---

# Delta Interpretation Rules

| Delta     | Meaning    |
| --------- | ---------- |
| 0.10–0.25 | Far OTM    |
| 0.40–0.55 | ATM        |
| 0.65–0.80 | Strong ITM |
| 0.90+     | Deep ITM   |

```text
Price ↑ + OI ↑
```

## Short Build-up

```text
Price ↓ + OI ↑
```

---

## Short Covering

```text
Price ↑ + OI ↓
```

---

## Long Unwinding

```text

```

# Smart Money Interpretation Rules

## Bullish Continuation

Conditions:

* CE short covering
* Delta rising
* Gamma high
* IV expanding

---

## Bearish Continuation

Conditions:

* PE short covering
* PE delta rising
* Gamma high

---

## Trap Warning

Conditions:

* Price breakout
* Delta flat
* Gamma weak
* IV weak
